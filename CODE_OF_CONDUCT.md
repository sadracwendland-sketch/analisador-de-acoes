# Código de Conduta

## Nosso compromisso

Para promover um ambiente aberto, respeitoso e colaborativo, nós nos comprometemos a tornar a participação neste projeto uma experiência livre de assédio para todas as pessoas, independentemente de idade, deficiência, etnia, identidade de gênero, nível de experiência, nacionalidade, aparência, religião ou orientação sexual.

## Padrões esperados

Exemplos de comportamento que contribuem para um ambiente positivo:

- usar linguagem respeitosa e profissional
- aceitar feedback construtivo
- focar no que é melhor para a comunidade e para o projeto
- demonstrar empatia com outros participantes
- manter discussões técnicas objetivas e civilizadas

Exemplos de comportamento inaceitável:

- ataques pessoais, assédio ou intimidação
- discurso de ódio ou discriminação
- publicação de informações privadas sem permissão
- comportamento ofensivo, sexualizado ou abusivo
- spam, trolling ou sabotagem intencional do projeto

## Responsabilidades dos mantenedores

Os mantenedores podem editar, remover ou rejeitar comentários, commits, issues, pull requests e outras contribuições que violem este código de conduta.

## Escopo

Este código de conduta se aplica a todos os espaços do projeto, incluindo issues, pull requests, discussões, documentação e canais associados.

## Reporte

Se você presenciar ou sofrer comportamento inadequado, abra um canal privado com os mantenedores. Se estiver usando o repositório publicamente, prefira reportar o incidente pelos contatos indicados em `SECURITY.md` ou no perfil do mantenedor.

## Consequências

Participantes que violem este código de conduta poderão receber advertência, bloqueio temporário ou remoção permanente da comunidade, conforme a gravidade do caso.
